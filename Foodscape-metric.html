Point pattern statistics

What’s the pOint in point pattern statistics?

Point patterns represent the xy location of objects or event occurrences in the space. Typically point patterns are classed in either of three groupings; clustered, regular and random (Nightingale, 2020). Also here: https://rpubs.com/loggaussian/areainteraction

image

Clustered point patterns

The clustered point pattern would consist of clusters of points with gaps in between. Additionally, a clustered point pattern can be considered to contain points which are situated closer to each other than would be expected for points which are situated/located completely at random.In fact, a point pattern with points that are situated completely at random are formally described as exhibiting “Complete Spatial Randomness” (CSR). Importantly, the “reference” point pattern which exhbits CSR and to which other point patterns can be compared to is the homogeneous Poisson point process. In terms of usage and interpretability, a clustered point pattern is thought to represent objects or events which tend to associate together. In particular some authors suggest that the points in such point patterns are representative of some degree of underlying “attraction” between the objects/events which they denote.

Regular point patterns

A regular point pattern, unlike the clustered, and random point patterns contains points which are distributed/situated in an ordered fashion. This distribution is more ordered than would be expected for points in a point pattern which exhibits CSR. The underlying interaction between objects/events denoted by the points in a regular point pattern is that of inhibition and described as a “negative” interaction.

Random point patterns

As described earlier, the point pattern which consists of a random distribution of points exhibits CSR and the homogeneous Poisson point process is an exemplar. The points in random point patterns are not dependent on each other, not ordered systemmatically, and there is no underlying interaction associated with their distribution.

Marked point patterns

For a given point pattern, each point represents one object/event. Additional information on each object (apart from its spatial location) may also be available. This additional information or ‘mark’ [@nightingale2019area; @baddeley2008analysing], associated with each point is considered to be an ‘attribute’ of that point. @baddeley2008analysing notes that a mark can be thought of as an additional coordinate for each point in the spatial pattern. A point pattern representing points which possess such additional information is classified as a marked point pattern. Examples of such point attributes are: tree diameter breast height, number of eggs in nest, weight of eggs, animal/plant species and COVID-19 positivity.

Note that marks can be qualitative/categorical such as species or quantitative/continuous such as average COVID-19 positivity. If the marks are qualitative, the marked point pattern is a multitype point pattern. This point pattern would be associated with different subpatterns, each representing one particular mark of the mark type. An example of this is a point pattern representing different plant species such that the mark is a quantitative discrete mark corresponding to the species of the plant.

Mathematically, for a given marked multitype point pattern \(\mathbf{x}\) in a bounded region in space \(W\), with \(n\) individual marks, there exists \(n\) subpatterns, \(x_{1:n}\) such that \(\mathbf{x} = {x_{1}, ...x_{n}}\). Finally, a mark can be multivariate in nature, such that each mark represents a list of attributes/variables relating to every point represented in the pattern.

The pair correlation function as a diagnostic tool

The pair correlation function (g(r)) allows us to view the spatial distribution of the points denoting the fast food outlets over different spatial scales. In particular, we can see whether the points are clustered, regular, or randomly distributed and over which spatial scales.

For a given fast food outlet, A, the pair correlation function, or g(r), estimates how likely one would find another fast food outlet a distance r from A relative to what would be obtained in a point pattern which exhitis complete spatial randomness (CSR). If g(r) is =1,greater than 1 or less than 1, we would conclude that the fast food outlets are distributed randomly at distance r, clustered at distance r and regular (displaying inhibition) at distance r, respectively.

The use of the pair correlation function to characterise the foodscapes regarding fast food outlets in Scotland is novel and we will illustrate its use with examples below.

Below it is clear that for SIMD Decile 1, there is spatial clustering of fast food outlets over a wide spatial range (multiple spatial scales) and in SIMD Decile 10, there is clustering at isolated and narrow spatial scales. This suggests an inequality in the food environment in these contrasting SIMD Deciles.
